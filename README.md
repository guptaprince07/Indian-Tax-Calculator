# 🇮🇳 India Income Tax Calculator (2026 Edition)

A clean and functional web application designed to calculate income tax based on the **New Tax Regime** for the Financial Year 2026-27.

## ✨ Features
* **Latest 2026 Rules:** Programmed with the ₹75,000 standard deduction and the updated tax slabs.
* **Section 87A Rebate:** Automatically calculates the zero-tax rebate for taxable income up to ₹12 Lakhs.
* **Quick-Select Buttons:** Easy buttons to instantly check taxes for 12L, 15L, 20L, 25L, and 30L.
* **Indian Currency Formatting:** Automatically adds commas (e.g., 15,00,000) as you type for better readability.
* **New Tab Results:** Generates a professional, easy-to-read tax report in a separate browser tab.

## 🛠️ Tech Stack
* **HTML5** - Structure and Layout.
* **CSS3** - Responsive design and styling.
* **Vanilla JavaScript** - Financial logic and currency formatting.

## 🚀 How to Use
1. Enter your Annual Gross Salary in the input field.
2. Or, click one of the "Quick Select" buttons for common salary figures.
3. Click **Calculate Tax** to view your detailed tax breakdown.
